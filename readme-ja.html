BIPARDY - ガイド＆完全ゲームメカニクス

🎯 ゲームコンセプト

たった2048の単語でビットコインの全セキュリティが守られていることを知っていましたか？ 🔐 BIPARDYは、BIP-39ニーモニック単語（暗号通貨シードフレーズの基盤）を中毒性のあるメンタルチャレンジに変換します。

推測する各単語は実際のウォレットリカバリ単語です - あなたは文字通りビットコインエコシステムを構築したレンガでプレイしています！ 🧱⚡

⚙️ 技術アーキテクチャ

  • 完全PWA: あらゆるデバイスにインストール、初回読み込み後オフライン動作
  • 暗号学的生成: crypto.getRandomValues()を使用した真のランダム性
  • アンチリピートシステム: 210単語バッファで最近の繰り返しを防止
  • 混合エントロピー: マウス移動、キーストローク、タッチイベントを組み合わせ
  • Fisher-Yatesアルゴリズム: 数学的に完璧なシャッフル
  • 完全レスポンシブ: あらゆる画面サイズで最適化

🎮 ゲームモード

🙋 ソロ

可能な限り最高の複合スコア構築に挑戦。プレッシャーはあなたの戦略的決断にかかっています。

👥 マルチプレイヤー（2-4プレイヤー）

ターン制戦術バトル。21単語に最初に到達した人が勝利するのではありません - 最高の複合スコアを構築した人が勝利します。

🎲 ゲームメカニクス

ラウンドフロー:

  1. 提示: BIP-39単語が現れる（最低5文字）
  2. 初期開示: 最初の4文字が自動表示
  3. 構築フェーズ: キーボードを使って残りの文字を完成させる
  4. 検証: 準備できたら解答を確認
  5. 評価: システムがポイント、ストリーク、進捗を計算

🎛️ 操作アーセナル

主要アクション:

  • 💡 HELP: 次の文字を順次表示（長さによる可変コスト）
  • ⏭️ SKIP: 現在の単語を放棄（ストリーク保持、カウンター増加、ポイント変化なし）
  • ✅ SUBMIT: 完成した答えをBIP-39ベースで検証
  • 🔍 トグル: 全長を表示/隠す（42%ボーナス犠牲）

テキスト入力:

  • 仮想キーボード: モバイルタッチ最適化
  • 物理キーボード: デスクトップで自動検出
  • オート大文字: 検証用に自動変換

🔢 完全スコアリングシステム

💎 基本リワード

  • 標準解決: 正解単語ごとに+21ポイント
  • 隠し長さボーナス: +9追加ポイント（42%エクストラ = 合計30）
  • 基本原理: 長さトグルは42%ボーナス犠牲

💸 ヘルプエコノミー（最大コスト制限）

哲学: コストは決して21ポイントの標準値を超えない

  • 5文字: 初回ヘルプ-21pts、それ以降FREE
  • 6文字: 1回目ヘルプ-10pts、2回目ヘルプ-11pts、それ以降FREE
  • 7文字以上: 各ヘルプ-10pts（制限なし）
  • 特別ペナルティ: ヘルプ付き5文字単語: -1pt追加

⚡ トランザクションタイミング

  • ヘルプコスト: 使用時に即座に控除
  • 解決ポイント: 正解確認時に付与
  • 純結果: ヘルプ乱用でマイナスになる可能性

🚫 懲罰システム

不正解:

  • 試行制限: 単語ごと2回の不正解
  • ペナルティ: 2回目エラー後-10ポイント
  • マルチプレイヤーで: ペナルティ後次プレイヤーに自動ターン

📊 プレイヤーメトリクス

  • 🎯 スコア: 純バランス（リワード - コスト - ペナルティ）
  • 🔥 ストリーク: 連続正解（エラーでゼロリセット、スキップで保持）
  • 📝 単語数: 完了総数（正解 + ヘルプ解決 + スキップ）

⚔️ マルチプレイヤープロトコル

🎯 ターンメカニクス

  • 循環ローテーション: 各単語後にプレイヤーが自動交替
  • ターン制限: プレイヤーごと単語ごと最大2回の不正解
  • 強制転送: エラーペナルティ後、次に自動ターン
  • ターン有効アクション: 解決、ヘルプ使用、スキップ、または失敗（最大2回）

🏁 終了条件

  • トリガー: 任意のプレイヤーが正確に21単語に到達
  • 即座終了: 保留中のターンは完了されない
  • 勝者決定: 複合スコアによる、到着順ではない

🏆 勝利アルゴリズム

勝利は最大複合スコアで決定 - 完成スピードではありません。

🧮 複合スコア公式

インテリジェント連結: スコア + ストリーク + 単語数を数値文字列として

ゼロ除外ルール: コンポーネント = 0は連結に含まれない

📊 例ケース:

🎯 プレイヤー1: 540pts、2ストリーク、21単語 → "540" + "2" + "21" = 540221
🎯 プレイヤー2: 176pts、19ストリーク、20単語 → "176" + "19" + "20" = 1761920
🏆 勝者: プレイヤー2 (1,761,920 > 540,221)

🔍 特殊ケース（ゼロ除外）:
• 211pts、0ストリーク、21単語 → "211" + "21" = 21121
• 0pts、0ストリーク、21単語 → "21" = 21
• 0pts、0ストリーク、0単語 → 0

🎨 この公式は戦略的緊張を生む - 高ストリークが高スコアを打ち負かせる
