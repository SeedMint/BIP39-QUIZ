🎲 BIPARDY

🔗 Open Source Code on GitHub 🔗

たった2048の単語でビットコインの全セキュリティが守られていることを知っていましたか？ 🔐 BIPARDYは、BIP-39ニーモニック単語（暗号通貨シードフレーズの基盤）を中毒性のあるメンタルチャレンジに変換します。

推測する各単語は実際のウォレットリカバリ単語です - あなたは文字通りビットコインエコシステムを構築したレンガでプレイしています！ 🧱⚡

🎯 ゲームコンセプト

BIPARDYは、ビットコインBIP-39単語リストから単語を完成させることに挑戦します。各単語の最初の4文字が表示されます - あなたのミッションは残りを推測することです。しかし、ここに落とし穴があります：すべての決定が重要です。ストリークを保持するためにスキップしますか？高価なヘルプを使用しますか？推測にすべてを賭けますか？選択はあなた次第です。

コアメカニクス:

  • 🎯 単語ソース: すべての単語は公式BIP-39単語リストから来ています（5文字以上の単語でフィルタリング）
  • 🔍 表示システム: 最初の4文字は常に表示されます
  • ⚡ 試行制限: 単語ごとに正確に2回の試行が与えられます
  • 🔥 ストリークメカニクス: 間違った答えはストリークを0にリセットしますが、スキップは保持します
  • 🏁 ゲーム終了条件: 12または24単語を完成させてゲームを終了します（BIP-39標準長）
  • 🎊 勝利条件: マルチプレイヤーでは、最も高い合計スコアが勝利します（最初に終了した人ではありません！）

🔢 完全スコアリングシステム

BIPARDYのスコアリングシステムは、正確性と戦略的リスクテイキングの両方を報います。すべてのアクションには結果があり、ポイント経済を理解することがゲームをマスターするために重要です。

📈 基本スコア:

  • 標準単語完成: +21ポイント
  • 隠し長さボーナス: +42%乗数 (21 → 30ポイントに変換)
  • ボーナス活性化: ラウンド全体を通じて単語長を隠しておく
  • ボーナス没収: 長さトグルを使用すると、その単語のみのボーナスが削除されます

🆘 ヘルプシステム経済学:

ヘルプシステムは、標準単語値を決して超えないように設計されたバランスの取れたコスト構造に従います:

  • 5文字単語: -21 points 最初のヘルプで、その後完全に無料
  • 6文字単語: -10 points 最初のヘルプ, -11 points 2番目のヘルプ、その後無料
  • 7+文字単語: -10 points per help （最大制限なし）
  • コスト上限: 5-6文字単語の場合、総ヘルプコストは21ポイントを決して超えません

⏱️ タイマーボーナスシステム:

  • クイックレスポンスボーナス: 各単語が明らかになったときに5秒タイマーが開始されます
  • ダイナミックボーナス: Up to +21 bonus points 即座の回答で
  • ボーナス減衰: ボーナスは5秒間で線形に減少し、タイマーが切れると0に達します
  • ボーナス停止: HELPを使用したり間違った推測をしたりすると、タイマーは即座に停止します
  • 最大ポテンシャル: 完璧な単語（30ベース）+ フルボーナス（21）= 単語あたり51ポイント

❌ ペナルティシステム:

  • 2つの間違った推測: -10ポイントペナルティ 即座に適用
  • ストリーク影響: 間違った推測はすべてストリークを0にリセットします
  • ターンメカニクス: マルチプレイヤーでは、2回の間違った推測の後、ターンが次のプレイヤーに渡ります
  • ユニバーサル適用: ペナルティはシングルプレイヤーとマルチプレイヤーモードの両方に適用されます

🔄 アクション結果:

  • スキップ: ストリークを保持し、ポイントを与えず、単語完成にカウントされます
  • ヘルプ: 進歩を保証し、ポイントがかかり、単語完成にカウントされます
  • 推測: リスク/リワード - 成功はストリークを構築してポイントを得て、失敗はストリークをリセット

🎮 ゲームコントロール

BIPARDYの戦略的ゲームプレイを進むために、これらの必須コントロールをマスターしてください:

  • HELP: シーケンスの次の文字を表示します。コストは単語の長さによって異なります（上記のヘルプシステム経済学を参照）。進歩を保証しますが、スコアを削減します。
  • SKIP: 現在の単語を完全にバイパスします。貴重なストリークを保持し、ゼロポイントを授与し、目標に向けて完了した単語としてカウントされます。
  • SUBMIT: 完成した単語推測を検証のために提出します。すべての欠けている文字を埋めたときにのみアクティブになります。
  • ゲーム長さトグル: 12または24単語ゲームを選択します（標準BIP-39ニーモニック長）。ゲーム開始前に設定。
  • 単語長トグル: 単語の総長を表示または隠します。完全な単語に含まれる文字数を表示しますが、その特定の単語の42％ボーナスを永続的に削除します。

📊 Player Statistics:

  • スコア: 正しい単語完成から獲得した総ポイントから、すべてのヘルプコストと間違った推測のペナルティを差し引いた値。これはあなたの合計スコアのベースコンポーネントです。
  • ストリーク: 現在の連続した正しい推測。間違った答えで0にリセットされますが、スキップはストリークを保持します。マルチプレイヤー合計スコアに重要。
  • 単語: 扱った単語の総数 - 成功した推測、ヘルプで完成した単語、スキップした単語を含みます。2回の間違った推測で失敗した単語は除外されます。

👥 マルチプレイヤーメカニクス

BIPARDYのマルチプレイヤーモードは、タイミング、リスク管理、スコア最適化が衝突する戦略的戦闘にゲームを変換します。

ターン構造:

  • ターン順: プレイヤーは順番に交代し、それぞれが自分の単語を取得します
  • 試行制限: 各プレイヤーは単語ごとに正確に2回の間違った推測を得ます
  • ターン完了: 2回の間違った推測の後、プレイヤーはペナルティを受け、ターンは即座に渡ります
  • 独立した進歩: 各プレイヤーの単語、スコア、ストリークは別々に追跡されます

ゲーム終了条件:

  • トリガー: ANY プレイヤーが目標単語数（12または24）に達すると、ゲームは即座に終了します
  • 公平性: すべてのプレイヤーは平等な機会を得ます - 誰もターンの途中で「カットオフ」されることはありません
  • 最終計算: 合計スコアはすべてのプレイヤーに対して同時に計算されます

🏆 勝利決定

ここがBIPARDYが真に戦略的になるところです: 勝者は合計スコアによって決定され、最初に終了した人ではありません！ この革命的なスコアリングシステムは、最初に目標単語数に達することが勝利を保証しない可能性があることを意味します。

🧮 強化された合計スコア公式:

合計スコア = スコア + ストリーク + 単語 (単一の数値として連結)

ゼロ省略ルール: 値が0のコンポーネントは、人為的な膨張を防ぐために連結から自動的に除外されます。

強化された合計スコア例:
🔵 プレイヤー1: 540ポイント, 2ストリーク, 24単語 = 540,224
🔴 プレイヤー2: 176ポイント, 19ストリーク, 23単語 = 1,761,923
🎉 勝者: プレイヤー2！ (より高い連結値)

ゼロ省略例:
211ポイント, 0ストリーク, 24単語 = 21,124 (ゼロストリーク省略)
0ポイント, 15ストリーク, 12単語 = 1,512 (ゼロポイント省略)
0ポイント, 0ストリーク, 24単語 = 24 (両方のゼロ省略)
🔴 プレイヤー2: 176ポイント, 19ストリーク, 23単語 = 1,761,923
🎉 勝者: プレイヤー2！ (より少ないポイントで最初に終了しなかったにもかかわらず)

境界事例の例:
211ポイント, 0ストリーク, 24単語 = 21,124 (ゼロストリーク省略)
0ポイント, 0ストリーク, 24単語 = 24 (両方のゼロ省略)
0ポイント, 0ストリーク, 0単語 = 0 (完全ゼロ状態)

🎯 戦略的影響:

  • ストリーク優先: 高いストリークは生のポイント合計を劇的に上回ることができます
  • ゴールライン・パラドックス: 目標単語数への急ぎは勝利を犠牲にする可能性があります
  • リスク管理: 時にはスキップがストリークをリスクにさらすよりも良いことがあります
  • Help vs. 推測: 戦略的なヘルプ使用は、大規模な合計スコアのためにストリークを保持できます

🧠 戦略的深度

BIPARDYは純粋な単語知識よりも戦略的思考を報います。ゲームの根本的な数学とリスク-リワード計算を理解することは、高度なプレイに不可欠です。

🎭 3つの道:

  • 🎯 推測者: ハイリスク、ハイリターン。大規模なストリークを構築するのに最適ですが、破滅的なリセットに脆弱です。
  • 🛡️ 計算者: コストを管理しながら進歩を保証する戦略的ヘルプ使用。より安全ですが、潜在的に上限が低い。
  • ⚡ スキッパー: ストリークを冷酷に保持し、巨大な合計スコアポテンシャルのために個別の単語ポイントを犠牲にします。

📐 数学的考慮事項:

  • 隠し長さボーナス: 42%乗数（9追加ポイント）は蓄積されると最終スコアに大きく影響する可能性があります
  • ヘルプコスト分析: 5文字単語の場合、保証された完成のために21ポイントを支払うことは、貴重なストリークを保持する価値があるかもしれません
  • ペナルティ回避: 間違った推測の10ポイントペナルティとストリーク損失は巨大な機会コストを生み出します
  • ストリーク指数価値: マルチプレイヤーでは、各ストリークポイントが合計スコア計算で指数関数的により価値のあるものになります

⚔️ マルチプレイヤー心理学:

  • ゴールライン錯覚: 目標単語数に急ぐ他のプレイヤーは、貧弱な合計スコアによってあなたに勝利を手渡すかもしれません
  • プレッシャー管理: 他のプレイヤーの進歩に基づいて計算されたリスクを取るか安全にプレイするかを知ること
  • リソース配分: あなたのポイントは限られています - 本当に重要な時にヘルプのために賢く使いましょう

あなたが遭遇する各単語は理論的に誰かの実際のビットコインウォレットシードフレーズの一部である可能性があります。あなたはただゲームをプレイしているのではありません - デジタルファイナンスの語彙をマスターしているのです！ 💎
